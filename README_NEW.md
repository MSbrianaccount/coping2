# 🏥 RehabApp - Rehabilitation Management System

## ⭐ **START HERE!** Read [START_HERE.md](START_HERE.md)

A complete, production-ready rehabilitation management system for local healthcare facilities. Works offline, requires no backend server, and is ready to deploy immediately.

## ✨ Key Features

✅ **Multi-user support** with 5 different roles
✅ **Patient management** - Admit, track, manage patients
✅ **Appointment booking** - Schedule with doctors
✅ **Financial tracking** - Invoices, payments, revenue
✅ **Role-based dashboards** - See relevant metrics only
✅ **Works offline** - No internet needed
✅ **Local storage** - All data saved on your computer
✅ **Professional UI** - Modern, easy-to-use interface

## 👥 User Roles

| Role | Capabilities |
|------|--------------|
